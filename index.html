ABOUT

Scan it and Hear it takes an image file as an input type and produces the audio file and text as an output extracted from the image.

This helps users of both usual and visually impaired to read a document. Users can use the extracted text for data entries to reduce time consumption in typing manually. The technologies used in Snap it and Hear it is Python , Flask Web Development , HTML , CSS
